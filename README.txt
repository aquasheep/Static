Introduction: 
This is Static, a game by Peter “aquasheep” Kennedy. 
The goal of the game is to create a beautiful, creative image through the evolution of random static with some basic tools applied. 
This game was designed as a submission for Ludum Dare 24: Evolution. 

Create,Evolve,Enjoy!
